Data Logger

A long time un-manned data logger network for irrigation system

  2014 to present

This is my student researcher Local data-logger network, for a PhD student in Land, Air and Water resources dept. at UC Davis.
This is a complete data-logging system that consisted of an coordinator and many data-logging nodes.
Each node handles various of sensors, collecting data every day, and send logs to coordinator via XBee. Coordinator will periodicly upload log files to the cloud via GPRS service.
